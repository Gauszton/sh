# 
 Automatyczny skrypt przygotowujący VM pod BugBounty
 
 Wymagania: 
  - użytkownik z uprawnieniami sudo 
  - [Na ubuntu] adduser nazwa; usermod -aG sudo nazwa;
 
 Instalacja:
  - git clone https://github.com/Gauszton/sh
  - cd sh/
  - chmod +x skrypt.sh
  - ./skrypt.sh
  
  
 Testowane 
  - Ubuntu 20.04 LTS
